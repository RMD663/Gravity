# Gravity
A 2D orbit simulator built using WebGL and p5.js.

## How to Run
Open the `index.html` file in a web browser.

## About the Project
This project was developed for a Web Programming class in my university program.  
Its objective was to create a real-time orbital simulation in the browser using WebGL.

## How to Create a Simulation

### Creating a Body
Create a body using the menu on the left.  
Enter the body's X and Y coordinates, radius, color, and mass, then click **Create**.

If a body has a mass greater than **20**, it becomes a **star** and does not receive initial inertia.

### Moving the Camera
Use the **arrow keys** to move the camera.

### Saving and Loading a Simulation
To save a simulation, enter a name in the input field and click **Save**.  
To load a simulation, type the simulation name and click **Load**.
